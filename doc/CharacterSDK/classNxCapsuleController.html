Detailed Description

A capsule character controller.

The capsule is defined as a position, a vertical height, and a radius. The height is the same height as for NxCapsuleShape objects, i.e. the distance between the two sphere centers at the end of the capsule. In other words:

p = pos (returned by controller)
h = height
r = radius

p = center of capsule
top sphere center = p.y + h*0.5
bottom sphere center = p.y - h*0.5
top capsule point = p.y + h*0.5 + r
bottom capsule point = p.y - h*0.5 - r
