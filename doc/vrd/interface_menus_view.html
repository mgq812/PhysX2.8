Transparent objects

When this is enabled, all non-selected objects will be transparent. If no objects are selected, all objects will be drawn opaque.


Show Contours

When this is enabled, the contours of all the objects will be drawn. This is quite heavy on big scenes, so turning it off most often doubles the framerate


Show Edges

When this is enabled, the edges of all the objects will be drawn. This is also an expensive operation and turning it off will result in faster rendering


Draw when unfocused

When this is disabled, the viewport will not update unless the main window is focused. When it is enabled, the viewport will always update when the underlying data changes.


Colour Scheme

Select what colour scheme to use in the viewport. Available modes are Default, Actor State and Hacker. In Default mode, each object have a random color and turn to a light blue if been picked; in Actor State mode, color of an object depends on the state of that object. Awake actors are green, sleeping actors are orange, kinematic actors are blue and static actors are red. In Hacker mode, all the objects will be in dark grey with a green outline.


Draw bounds

Toggles whether object bounds should be drawn or not. Available modes are on, off and selected. In selected mode, bounds will only be drawn on objects that are selected.


Draw contacts

Toggles whether contacts should be drawn or not. Available modes are on, off and selected. In selected mode, only contacts that act upon a selected object will be drawn.
Note: In each single frame, PVD only can get no more than 32767 contacts.


Draw frames

Toggles whether object frames should be drawn or not. Available modes are on, off and selected. In selected mode, framess will only be drawn on objects that are selected.


Draw mass centers

Toggles whether mass centers should be drawn or not. Available modes are on, off and selected. In selected mode, mass centers will only be drawn on objects that are selected.


Draw joints

Toggles whether joints should be drawn or not. Available modes are on, off and selected. In selected mode, only joints connected to a selected object will be drawn.


Draw velocities

Toggles whether object velocities should be drawn or not. Available modes are on, off and selected. In selected mode, velocities will only be drawn on objects that are selected.


Draw force fields

Toggles whether force fields should be drawn or not. Available modes are on and off.


Draw grid

Toggles whether a grid should be drawn at the origin. The grid may be useful as a navigation landmark in small scenes no or few static objects.


Up axis

Sets the axis to use as up axis in the viewport. Available modes are x+, x-, y+, y-, z+ and z-. This affects nothing but viewport navigation.


Camera

Selects what type of camera to use in the viewport. Available modes are Orbit, Fly and any custom cameras found in the stream. Orbit and Fly affects viewport navigation. Selecting a custom camera will lock the view to the selected camera.


Coordinate System

Selects left-handed or right-handed coordniate system you want to use in the viewport. Acailable modes are left-handed and right-handed.
