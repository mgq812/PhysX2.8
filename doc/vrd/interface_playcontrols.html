Interface - Views

The Visual Remote Debugger interface consists of a number of windows, or views. These views all interpret the same data, the scene, but displays it in different ways. The different types of views are:

Event List
Play Controls
Profiler
Scene Browser
Viewport



Play Controls
While not a view in the same sense as the others, the play controls are central to working with the debugger. These are used to control the playback of the stream. The available buttons are:

Stop
Stops playback of the stream, allowing for closer inspection of a certain state.
        Shortcut: Shift + Space / Space
Play backwards/forwards
Plays the stream forwards or backwards in realtime.
        Shortcut: Shift + Space / Space
Step backwards/forwards
Steps one frame/timestep forwards or backwards, and then stops.
        Shortcut: Arrow left / Arrow right
Go to beginning/end
Skips through the whole stream to the beginning or end.
        Shortcut: Home / End
Crop backwards/forwards
Discard data before/after current frame.


Below the playback buttons there is a time slider. A gray bar always shows the current position in the stream, that is, if the bar looks like this
the debugger has played about a third through the stream. Clicking anywhere in the time slider will make the play position jump there. Note that this will not snap to the nearest framebreak.
