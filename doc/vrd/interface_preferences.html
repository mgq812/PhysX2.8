Interface - Preferences

The preferences window is where global options for the applications can be set. The currently available options are as follows.



Scene settings

Scene Scale
Use this to scale the Remote Debugger Settings to suit the scale of your scene. Picking is affected by this setting. If you are using a small or large scene and experience picking issues, adjust this setting according to your scene scale. If the Debugger is running a local simulation the simulation settings are affected. See below on how. The camera behavior is also affected by this setting.

Gizmo Scale
Determines the scale of the symbols drawn in the viewport.

Camera speed
Scales the speed of the camera when in fly mode. The camera speed is also affected by the Scene Scale.

Culling Far Distance
Set the distance of far clipping plane.

Transparency
Determines the transparency of the objects in viewport, the value must be in the range [0, 1]. This is only effect when Transparent objects in view menu is checked.

Use Vertex Buffer Objects
When this option is checked, all vertex buffers will be copied to the graphics card for faster rendering. This comes with potentially more graphics card memory being used by VRD.





Network

Listen port
Determines the port that the debugger listens on for new connections.
