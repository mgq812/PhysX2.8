New... Ctrl-N
Open... Ctrl-O

Preferences...    Ctrl-P
Exit... Ctrl-Q


New...

Clears all loaded data, and puts the debugger in listening mode, waiting for incoming connections.


Open...

Opens an existing PXD file. When opening an event dump (PXD), interaction will not be available and the debugger's functionality is limited to regular playback.


Save As...

Saves the current state as a PXD or a gzip compressed PXD (select .pxd.gz as file extension)


Preferences...

Opens a Preferences dialog, where various general options can be set.


Exit...

Closes the debugger, discarding any loaded unsaved data.
