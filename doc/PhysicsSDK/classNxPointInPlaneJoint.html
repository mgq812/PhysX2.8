Detailed Description

A point in plane joint constrains a point on one body to only move inside a plane attached to another body.

The starting point of the point is defined as the anchor point. The plane through this point is specified by its normal which is the joint axis vector.

Creation

NxPointInPlaneJointDesc pipDesc;

pipDesc.actor[0] = actor0;
pipDesc.actor[1] = actor1;

pipDesc.setGlobalAnchor(globalAnchor); //point attached to plane
pipDesc.setGlobalAxis(globalAxis); //point to attach

pipDesc.jointFlags |= NX_JF_COLLISION_ENABLED; // enable collision detection between each actor.

NxPointInPlaneJoint *pipJoint=(NxPointInPlaneJoint*)gScene->createJoint(pipDesc);
