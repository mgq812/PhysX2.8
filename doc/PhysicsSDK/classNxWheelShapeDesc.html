NxSpringDesc NxWheelShapeDesc::suspension
 

data intended for car wheel suspension effects.

targetValue must be in [0, 1], which is the rest length of the suspension along the suspensionTravel. 0 is the default, which maps to the tip of the ray cast.

Range: .spring [0,inf)
Range: .damper [0,inf)
Range: .targetValue [0,1]
Default: See NxSpringDesc

See also:
NxSpringDesc NxWheelShape.setSuspension() NxWheelShape.getSuspension()

NxReal NxWheelShapeDesc::suspensionTravel
