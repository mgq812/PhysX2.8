Detailed Description

Class used to retrieve limits(e.g. max number of bodies) for a scene. The limits are used as a hint to the size of the scene, not as a hard limit (i.e. it will be possible to create more objects than specified in the scene limits).

Platform:

  • PC SW: Yes
  • PPU : Yes
  • PS3 : Yes
  • XB360: Yes
