virtual bool NxClothMesh::saveToDesc (  NxClothMeshDesc &  desc  )  const [pure virtual]
 

Saves the cloth descriptor. A cloth mesh is created via the cooker. The cooker potentially changes the order of the arrays references by the pointers points and triangles. Since saveToDesc returns the data of the cooked mesh, this data might differ from the originally provided data. Note that this is in contrast to the meshData member of NxClothDesc, which is guaranteed to provide data in the same order as that used to create the mesh.
