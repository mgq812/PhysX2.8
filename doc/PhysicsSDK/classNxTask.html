Detailed Description

Interface to an SDK task.

The SDK submits NxTasks to users custom scheduler for execution on an appropriate thread.

virtual void NxTask::execute (   )  [pure virtual]
 

Execute the task supplied by the SDK.

This method can be called from an arbitary thread. Note however that each additional thread calling this function will incur a memory overhead for thread specific data.
