NxReal NxRevoluteJointDesc::projectionDistance
 

The distance beyond which the joint is projected.

projectionMode is NX_JPM_POINT_MINDIST, the joint gets artificially projected together when it drifts more than this distance. Sometimes it is not possible to project (for example when the joints form a cycle) Should be nonnegative. However, it may be a bad idea to always project to a very small or zero distance because the solver *needs* some error in order to produce correct motion.

Range: [0,inf)
Default: 1.0
