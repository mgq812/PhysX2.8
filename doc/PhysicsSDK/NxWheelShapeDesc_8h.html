Enumeration Type Documentation

enum NxWheelShapeFlags
 

Enumerator:
NX_WF_WHEEL_AXIS_CONTACT_NORMAL  Determines whether the suspension axis or the ground contact normal is used for the suspension constraint.
NX_WF_INPUT_LAT_SLIPVELOCITY  If set, the laterial slip velocity is used as the input to the tire function, rather than the slip angle.
NX_WF_INPUT_LNG_SLIPVELOCITY  If set, the longutudal slip velocity is used as the input to the tire function, rather than the slip ratio.
NX_WF_UNSCALED_SPRING_BEHAVIOR  If set, does not factor out the suspension travel and wheel radius from the spring force computation. This is the legacy behavior from the raycast capsule approach.
NX_WF_AXLE_SPEED_OVERRIDE  If set, the axle speed is not computed by the simulation but is rather expected to be provided by the user every simulation step via NxWheelShape::setAxleSpeed().
NX_WF_EMULATE_LEGACY_WHEEL  If set, the NxWheelShape will emulate the legacy raycast capsule based wheel. See NxTireFunctionDesc.
NX_WF_CLAMPED_FRICTION  If set, the NxWheelShape will clamp the force in the friction constraints. See NxTireFunctionDesc.
