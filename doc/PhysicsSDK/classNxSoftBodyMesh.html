virtual bool NxSoftBodyMesh::saveToDesc (  NxSoftBodyMeshDesc &  desc  )  const [pure virtual]
 

Saves the soft body mesh descriptor. A soft body mesh is created via the cooker. The cooker potentially changes the order of the arrays references by the pointers vertices and triangles. Since saveToDesc returns the data of the cooked mesh, this data might differ from the originally provided data. Note that this is in contrast to the meshData member of NxSoftBodyDesc, which is guaranteed to provide data in the same order as that used to create the mesh.
