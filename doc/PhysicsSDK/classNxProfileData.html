Detailed Description

Array of profiling data.

profileZones points to an array of numZones profile zones. Zones are sorted such that the parent zones always come before their children. Some zones have multiple parents (code called from multiple places) in which case only the relationship to the first parent is displayed. returned by NxScene::readProfileData().
