NxReal NxBodyDesc::sleepAngularVelocity
 

Maximum angular velocity at which body can go to sleep.

If negative, the global default will be used.

Setting the sleep angular/linear velocity only makes sense when the NX_BF_ENERGY_SLEEP_TEST is not set. In version 2.5 and later a new method is used by default which uses the kinetic energy of the body to control sleeping.

Range: [0,inf)
Default: -1.0

Platform:

  • PC SW: Yes
  • PPU : Yes
  • PS3 : Yes
  • XB360: Yes
See also:
NxActor.setSleepAngularVelocity() NxActor.getSleepAngularVelocity() sleepLinearVelocity
