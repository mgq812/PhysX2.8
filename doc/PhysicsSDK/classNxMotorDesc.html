Detailed Description

Describes a joint motor.

Some joints can be motorized, this allows them to apply a force to cause attached actors to move.

Joints which can be motorized:

  • NxPulleyJoint
  • NxRevoluteJoint

NxJointDriveDesc is used for a similar purpose with NxD6Joint.

Example (for a revolute joint):

NxRevoluteJointDesc revoluteDesc;

revoluteDesc.actor[0] = actor0;
revoluteDesc.actor[1] = actor1;

revoluteDesc.setGlobalAxis(NxVec3(1,0,0)); //rotate around the x axis.
revoluteDesc.setGlobalAnchor(NxVec3(0.0f,10.0f,0.0f); //anchor, point the axis passes through.

revoluteDesc.motor.maxForce = 1000;
//The joint tries to maintain a velocity of 0.15, but can only apply up to maxForce
revoluteDesc.motor.velTarget = 0.15; 
revoluteDesc.flags |= NX_RJF_MOTOR_ENABLED; //Make this a motorized joint.

NxRevoluteJoint* driveJoint = (NxRevoluteJoint*) gScene->createJoint(revoluteDesc));

NX_BOOL NxMotorDesc::freeSpin
 

If true, motor will not brake when it spins faster than velTarget.

Default: false

NxReal NxMotorDesc::maxForce
 

The maximum force (or torque) the motor can exert.

Zero disables the motor. Default is 0, should be >= 0. Setting this to a very large value if velTarget is also very large may cause unexpected results.

Range: [0,inf)
Default: 0.0

NxReal NxMotorDesc::velTarget
 

The relative velocity the motor is trying to achieve.

The motor will only be able to reach this velocity if the maxForce is sufficiently large. If the joint is spinning faster than this velocity, the motor will actually try to brake(see freeSpin).

If you set this to infinity then the motor will keep speeding up, unless there is some sort of resistance on the attached bodies. The sign of this variable determines the rotation direction, with positive values going the same way as positive joint angles.

Default is infinity.

Range: [0,inf]
Default: NX_MAX_REAL
