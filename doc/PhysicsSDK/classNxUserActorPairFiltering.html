Callback to allow the user to decide whether to filter a certain actor pair.

Use the actor member of the NxActorPairFilter objects to decide whether to filter out the contact between the pair. Set the filtered member to true to apply filtering.

Parameters:
filterArray An array of actor pairs for which filtering is to be decided.
arraySize The number of elements in filterArray.
