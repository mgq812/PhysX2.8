Detailed Description

Descriptor class for NxCompartment. A compartment is a portion of the scene that can be simulated on a different hardware device than other parts of the scene.

 Thread affinity mask for the compartment thread. Defaults to 0 which means the SDK determines the affinity.
NxReal timeScale
 It is possible to scale the simulation time steps of this compartment relative to the primary scene using this nonnegative value. A scale of zero does not simulate the compartment. The default value is 1.0f.
