Detailed Description

3 Element vector class.

This is a vector class with public data members. This is not nice but it has become such a standard that hiding the xyz data members makes it difficult to reuse external code that assumes that these are public in the library. The vector class can be made to use float or double precision by appropriately defining NxReal. This has been chosen as a cleaner alternative to a template class.
